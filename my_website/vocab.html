This is my Vocab Page!

  1. Sprites: It's a graphic object that is able to move it to the top or behind the background.
  2. Script: It is a a set of instruction where it tells the sprite or the stage what to do.
  3. Iteration: is where you repeat a procedure multiple times.
  4. Stage: The place in which all of the sprites move.
  5. Costumes: Are the different appearances that a spirit would have.
  6. Sounds: The noises that are created in the program.
  7. Pseudocode: A simple way of writing program codes.
  8. Pair program: Where two people take turns on who is leading the program on a single computer.
  9. Integrated Development Environment(IDE): It is where the developer can do anything for the program such as making it, running it, and debugging in one application.
  10. See inside: Is a button on scratch in where you are able to see other people's work.
  11. Boolean Expression: It is a statement in which it tells if it's true or false mostly are in the if structure.
  12. Execute: It is to run a program or just an instruction.
  13. Bug: Where a part of a program that makes it to make an error or a different outcome.
